# RealtyLT — Presentation Suite

Four self-contained, premium deliverables produced in the final Fable audit pass (2026-07-12). Each is a
single HTML file (no external assets — CSP-safe, works offline) and was published as a claude.ai Artifact.
They share one brand system (navy/azure, monospace utility face, blueprint motif) but each has its own job.

| File | What it is | Use it for |
|---|---|---|
| `architecture.html` | **Architecture deep-dive** — the whole system end to end, with diagrams (IDX replication + on-demand photo proxy, the CRM automation engine, the AI journey, the agent loop, integrations, security, deployment). | Answering any technical "how does it work" question. |
| `portfolio.html` | **Capabilities / portfolio deck** — what was built and its value, framed for both employers and business clients. | Job applications; pitching services to businesses. |
| `qa-prep.html` | **Interview / client Q&A prep** — the likely technical/product/business questions (and the skeptical ones), each with a strong, honest answer grounded in the actual build. | Prepping for an interview or a client call. |
| `one-pager.html` | **One-page overview** — the whole suite at a glance: what it is, the three apps, the stack, the exact status. | A leave-behind / quick intro. |

**To view:** open any file in a browser, or re-publish via the Artifact tool. Each has a light/dark theme
toggle and is mobile-responsive.

**Source of truth:** the content is drawn from the live systems and the audit — `ARCHITECTURE.md`,
`AGENT_LEARNINGS.md`, `docs/MLS-INTEGRATION.md`, `docs/crm-audit/CRM-AUDIT.md`, and `docs/OWNER-GO-LIVE.md`.
Every number (5,362 listings, 6 counties, 23 RLS tables) and every claim is verified, and the honest gaps
are stated, not hidden.
